<objective>
Wire the graph to persistent state: add a SqliteSaver-backed checkpointer, write pre-narration and final-turn checkpoints in the prescribed order, implement first-class graph interrupts for `/pause`, `/line`, `/retcon`, budget hard-stop, and session-end, resume at the next prompt when a final checkpoint exists, and record every agent activation for audit. Delivers GRAPH-02, GRAPH-03, GRAPH-04, GRAPH-05, and AI-12.

Purpose: Without persistent checkpoints and first-class interrupts, the TUI's Phase 3 command stubs cannot achieve the crash-safe turn lifecycle required by `PERSISTENCE_SPEC.md §3-§5` or the interrupt semantics required by `GAME_SPEC.md §8-§9`. This is the glue that makes the graph usable by the existing TUI shell.
Output: Persistent graph builder, interrupt dispatcher, activation logger, migration 0005, rewired `/pause` and `/line` commands that post graph interrupts instead of only writing safety events.
</objective>

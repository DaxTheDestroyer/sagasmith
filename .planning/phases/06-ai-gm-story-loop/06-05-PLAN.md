## Overview

Create a minimal MemoryPacket implementation for Phase 6 that provides basic context without full memory assembly. This allows Oracle and Orator to have some contextual awareness while deferring full memory implementation to Phase 7.

### Step 1: Enhance Memory Packet Assembly Skill

**File:** `src/sagasmith/agents/archivist/skills/memory-packet-assembly/SKILL.md`

Update the existing skill definition with:
- Procedure for stub memory packet assembly
- Input requirements: scene context, recent transcript
- Output requirements: MemoryPacket within token cap
- Success signal: Packet size never exceeds cap, recent context available

### Step 2: Implement Memory Packet Stub Logic

**File:** `src/sagasmith/agents/archivist/skills/memory-packet-assembly/logic.py`

Create the stub implementation that:
- Retrieves recent transcript entries from SQLite
- Creates minimal entity references
- Constructs bounded summary context

- Returns valid MemoryPacket object

### Step 3: Implement Transcript Context Retrieval

**File:** `src/sagasmith/agents/archivist/transcript_context.py`

Create logic to retrieve recent transcript context:
- Query SQLite for recent turn records

### Step 4: Implement Entity Reference Stubbing

**File:** `src/sagasmith/agents/archivist/entity_stubs.py`

Create logic for minimal entity reference handling:
- Stub common entity types (NPCs, locations, items)
- Create provisional references for mentioned entities
- Maintain consistent entity ID format

### Step 5: Update Archivist Node

**File:** `src/sagasmith/agents/archivist/node.py`

Modify the archivist_node function to:
- Replace any existing memory assembly with stub implementation
- Call memory packet assembly skill
- Store MemoryPacket in state
- Add proper skill activation logging

### Step 6: Integrate with Oracle and Orator

**File:** `src/sagasmith/agents/oracle/node.py`

- Pass MemoryPacket to scene brief composition logic

**File:** `src/sagasmith/agents/orator/node.py`

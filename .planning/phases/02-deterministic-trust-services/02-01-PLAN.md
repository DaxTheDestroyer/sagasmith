## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| process → OS keyring | `keyring.get_password` crosses out to the OS credential store; return value is plaintext secret. |
| process → env | `os.environ[...]` read; env var value is plaintext secret. |
| raised exception → log/checkpoint/vault | Any exception reaching user-facing text or persisted artifacts must not carry the secret value. |
| persisted/exported artifact → user | Any text surface scanned by `RedactionCanary` must flag secret-shaped substrings. |
